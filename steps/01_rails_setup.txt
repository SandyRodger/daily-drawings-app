6. Add routes

```

  namespace :api do
    resources :drawings, only: [:index, :show, :create]
  end
end
```

, which corresponds to the following APIs:
  - GET /api/drawings
  - GET /api/drawings/:id
  - POST /api/drawings

7. Add controller actions

open `app/controllers/api/drawings_controller.rb`

replace contents with:
```
class Api::DrawingsController < ApplicationController
  def index
    drawings = Drawing.order(created_at: :desc)
    render json: drawings
  end

  def show
    drawing = Drawing.find(params[:id])
    render json: drawing
  end

  def create
    drawing = Drawing.create!(drawing_params)
    render json: drawing
  end

  private

  def drawing_params
    params.require(:drawing).permit(:title, :date, :caption, :notes, :image_url)
  end
end
```

8. Start the server

`rails server`
